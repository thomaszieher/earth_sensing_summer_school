Querying ERA5-data:
1) Register at https://cds.climate.copernicus.eu
2) Follow the instructions on https://cds.climate.copernicus.eu/how-to-api:
	- prepare the file .cdsapirc with the following content:
		url: https://cds.climate.copernicus.eu/api
		key: <add your personal api key here>
	- Copy the .cdsapirc file to your home folder
3) Install the cdsapi library
